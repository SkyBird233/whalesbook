# whalesbook


## Project Setup

Install dependencies:
```sh
pnpm install
```

Generate client:
> With backend running on localhost:8000
```sh
pnpm openapi-ts
```

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Type-Check, Compile and Minify for Production

```sh
pnpm build
```

### Lint with [ESLint](https://eslint.org/)

```sh
pnpm lint
```
